
## Overview
This Power BI dashboard provides comprehensive HR analytics insights, helping organizations monitor workforce demographics, employee distribution, and organizational trends.

## Objectives
- Analyze employee headcount trends
- Monitor gender diversity
- Track workforce distribution by country
- Understand age demographics
- Evaluate business unit performance
- Analyze job role distribution

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel

## Dashboard Features

### Key Metrics
- Total Employee Headcount
- Female Employee Percentage
- Male Employee Percentage

### Visualizations
- Employee Headcount Volume Trend
- Employee Headcount by Gender
- Employee Age Distribution
- Employee Headcount by Country
- Employee Headcount by Business Unit
- Employee Headcount by Job Role

### Interactive Filters
- Date
- Business Unit
- Country
- Employee Name
